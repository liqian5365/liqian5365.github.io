css梯形实现

之前复习css的时候在网上看了一下用css来实现各种图形，什么圆形，三角形，梯形等等。
当我们设计把UI稿给我的时候大概看了一下，还好，应该不会很难。

然后整个写样式的流程大概是这样的：
1.首先温习一下三角形的实现：


2.再变个形，就是个实心梯形了。好了，梯形可以是实现了。开心。


一个实心的梯形。哭。
好像不太满足需求。
终于在各种帮助下找到了一种可以实现空心梯形的方法：
transform: skew()
先来看下效果：


很接近设计稿了。
但是设计稿上，梯形里面是有图片的。
放个图片试试效果。

但是图片里面的金鱼好像有点变形了。
然后给图片做一个变形，和外部梯形反方向。图片形状正常了一些。

做到这一步，整个UI图比较难的部分就已经完成了。
之后再更新没有想起来的点吧。
